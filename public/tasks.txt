Приложение новостей с подробностями
Создайте React-приложение, которое будет отображать список новостей.
При клике на новость, подробное содержание этой новости должно появляться. 
Для управления состоянием новостей используйте Redux.

Приложение для создания статей новостей
Разработайте React-приложение, которое позволит пользователям создавать статьи новостей. 
Статьи должны сохраняться в локальном хранилище и отображаться в списке. 
Используйте Redux для управления статьями новостей и обновления списка.

Приложение для управления списком задач
Созайте React-приложение, которое даст пользователям возможность управлять своим списком задач. 
Задачи должны сохраняться в локальном хранилище и отображаться в списке. 
Используйте Redux для управления задачами и обеспечения плавного обновления списка.
